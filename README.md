# react-native-magic-orb

A Skia and Reanimated based shader, creating a Apple like Ai Orb/Sphere

## Installation


```sh
npm install react-native-magic-orb
```


## Usage


```js
import { multiply } from 'react-native-magic-orb';

// ...

const result = multiply(3, 7);
```
